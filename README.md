# FoxConnect Documentation

The FoxConnect documentation can be viewed at [docs.foxconnect.app](https://docs.foxconnect.app)

FoxConnect is a LINE‑First Booking & CRM Platform for Service Businesses.   
FoxConnect turns Thailand’s over 50 million LINE users into a ready‑to‑book customer base, letting salons, clinics, fitness studios and other service providers accept appointments, manage staff schedules and nurture relationships – all inside the familiar LINE messenger interface.

FoxConnect's homepage can be found at [foxconnect.app](https://foxconnect.app)
