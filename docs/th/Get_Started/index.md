# เริ่มต้นใช้งาน


หากคุณเพิ่งเริ่มใช้ FoxConnect ให้เริ่มต้นด้วยคู่มือเริ่มต้นใช้งานฉบับย่อนี้ คู่มือนี้จะแนะนำขั้นตอนที่จำเป็นที่สุด — เมื่อดำเนินการครบแล้ว ระบบการจองของคุณจะพร้อมใช้งานอย่างสมบูรณ์

หลังจากตั้งค่าฟังก์ชันหลักแล้ว คุณสามารถสำรวจคุณสมบัติเพิ่มเติม เช่น:

- **Branches** – จัดการหลายสาขา  
- **Staff** – กำหนดพนักงานให้กับการนัดหมาย  
- **Loyalty** – ใช้งานระบบรางวัลและสิทธิพิเศษ

แต่ละหัวข้อเหล่านี้มีส่วนเฉพาะที่ครอบคลุมรายละเอียดในส่วนถัดไป
