# เพิ่มพนักงานของคุณ

**การเพิ่มสมาชิกพนักงานใน FoxConnect**

1. **เปิดส่วน Staff**  
   - จากเมนูด้านซ้าย คลิก **Staff**

2. **สร้างโปรไฟล์พนักงานใหม่**  
   - คลิกปุ่ม **Add Staff** ที่มุมบนขวา  
   - กรอกข้อมูลในฟิลด์ที่จำเป็น:

| ฟิลด์ | หมายเหตุ |
|-------|-------|
| **First & Last Name** | จำเป็น |
| **Email Address** | ไม่บังคับ ปล่อยว่างหากไม่มี |
| **Phone Number** | ไม่บังคับ ปล่อยว่างหากไม่มี |
| **Description** | สำหรับใช้งานภายในเท่านั้น — ข้อความนี้จะไม่แสดงให้ลูกค้าเห็น |
| **Image URL** | ลิงก์ไปยังรูปภาพที่จะแสดงบนหน้าการจองเมื่อลูกค้าเลือกพนักงานคนนี้ |

3. **กำหนดสาขา**  
   - ในดรอปดาวน์ **Branch** เลือกทุกสาขาที่พนักงานคนนี้จะให้บริการ  
   - สามารถเลือกได้หลายรายการ โดยทุกสาขาที่เพิ่มไว้ใน FoxConnect จะปรากฏในรายการ

4. **กำหนดบริการ**  
   - ในดรอปดาวน์ **Select Services** เลือกบริการหนึ่งรายการหรือมากกว่าที่พนักงานคนนี้สามารถให้บริการได้  
   - **สำคัญ:** พนักงานจะปรากฏในขั้นตอนการจองของลูกค้าก็ต่อเมื่อมีการกำหนดบริการอย่างน้อยหนึ่งรายการเท่านั้น

5. **ตั้งชั่วโมงทำงาน**  
   - สลับไปที่แท็บ **Working Hours** และระบุตารางเวลาส่วนตัวของพนักงาน  
   - เวลาเหล่านี้จะ *แทนที่* ชั่วโมงทำการทั่วไปที่ตั้งไว้ใน Booking Settings เพื่อให้มีความพร้อมใช้งานเฉพาะบุคคล  
   - ช่วงเวลาที่ได้จะเป็นสิ่งที่ลูกค้าเห็นเมื่อจองพนักงานคนนี้

6. **เพิ่มวันหยุด**  
   - ในแท็บ **Days Off** กรอกวันหยุดหรือช่วงพักเพิ่มเติมที่เฉพาะเจาะจงสำหรับพนักงานคนนี้  
   - วันที่เหล่านี้จะรวมกับวันหยุดของธุรกิจโดยรวม เพื่อป้องกันการจองซ้ำซ้อน

7. **บันทึกและตรวจสอบ**  
   - คลิก **Save** เพื่อสร้างโปรไฟล์พนักงาน  
   - พนักงานใหม่จะปรากฏในรายการ การคลิกชื่อของพวกเขาจะเปิดหน้ารายละเอียดที่แสดง:

   - ข้อมูลส่วนตัว  
   - บริการและสาขาที่ได้รับมอบหมาย  
   - การจองที่กำลังจะมาถึง (มองเห็นได้บนปฏิทินพนักงานและปฏิทิน FoxConnect หลัก)

เมื่อมีการจอง นัดหมายของพนักงานจะเติมทั้งสองปฏิทินโดยอัตโนมัติ เพื่อให้การจัดตารางเวลาสอดคล้องกันทั่วทั้งแพลตฟอร์ม
