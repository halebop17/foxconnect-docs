# เพิ่มสาขาของคุณ

**การเพิ่มสาขาใหม่ใน FoxConnect**

หากธุรกิจของคุณดำเนินการจากสถานที่เดียว คุณสามารถข้ามส่วนนี้ได้ หากไม่ใช่ ให้ทำตามขั้นตอนเหล่านี้เพื่อสร้างสาขาเพิ่มเติม:

1. **ไปที่ Branches**  
   - ในเมนูด้านซ้าย คลิก **Branches** เพื่อเปิดหน้าจัดการสาขา

2. **สร้างสาขาใหม่**  
   - คลิกปุ่ม **New Branch** ที่มุมบนขวา  
   - ในป็อปอัป กรอกข้อมูลในฟิลด์ที่มีให้:

| ฟิลด์ | วัตถุประสงค์ |
|-------|---------|
| **Name** | ชื่อสาขา (จำเป็น) |
| **Address** | ที่ตั้งทางกายภาพของสาขา |
| **Phone Number** | หมายเลขติดต่อหลักของสาขา |
| **Email** | ที่อยู่อีเมลเฉพาะของสาขา (ไม่บังคับ) |
| **Map URL** | ลิงก์ไปยังแผนที่ (เช่น Google Maps) ที่จะแสดงบนหน้าการจอง เพื่อให้ลูกค้าดูสถานที่ก่อนยืนยันนัดหมาย |
| **Description** | ข้อความสั้น ๆ ที่ไม่บังคับ แสดงให้ลูกค้าเห็นเมื่อเลือกสาขานี้ |
| **Image URL** | ลิงก์ไปยังรูปภาพที่แสดงถึงสาขา แสดงบนหน้าจองการจอง |

3. **ตั้งชั่วโมงทำการ**  
   - สลับแต่ละวันในสัปดาห์เปิดหรือปิดตามตารางเวลาของสาขา  
   - กรอกเวลาเปิดและปิด รวมถึงช่วงพักใด ๆ

4. **บันทึกสาขา**  
   - คลิก **Create Branch** เมื่อกรอกข้อมูลในฟิลด์ที่จำเป็นทั้งหมดแล้ว  
   - สาขาใหม่จะปรากฏในรายการและสามารถกำหนดพนักงานและบริการได้

หลังจากสร้างสาขาแล้ว คุณสามารถกำหนดค่ารายชื่อพนักงาน รายการบริการ และการตั้งค่าอื่น ๆ เฉพาะของสาขานั้นได้ในส่วนถัดไป
