# Get Started


If you’re new to FoxConnect, begin with this quick‑start guide. It walks you through the most essential steps—once completed your booking system will be fully operational.

After setting up the core functionality, you can explore additional features such as:

- **Branches** – Manage multiple locations.  
- **Staff** – Assign employees to appointments.  
- **Loyalty** – Implement rewards and incentives.

Each of these topics is covered in dedicated sections that follow.
